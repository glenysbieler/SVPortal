This folder contains LSL code for the optional in world object that will communicate with the website and relay some messages inworld. 
When an offline is sent - the website simply adds the entry into the im_offline table.  This works really well if the agent is offline.  
However if they are currently offline they wont get that message until NEXT time they login.

This script provides an optional in world notification (although the IM will come from the OBJECT not an AVATAR).

to use this - make an object and put this script in.   Be sure to leave this object somewhere safe where it will be always running.
